Learning Paths

Certified Cloud Professional

Get started with basic knowledge about Exoscale. Build a comprehensive knowledge base about our product portfolio and the cloud.

Certified Solution Architect

Deep-dive into the Exoscale cloud platform. Learn how to automate cloud infrastructure, leverage our features and get an unmatched understanding of our product range.

Certified Container Engineer

Building up basic K8s knowledge, and then we dive deeper into Kubernetes concepts, related storage technologies, applications and troubleshooting.

INTRO Kubernetes

Understand the basic principles of containers and container orchestration, as well as why Kubernetes and containers are beneficial.

INTRO DBaaS

Build your basic understanding of database concepts, benefits and how to select appropriate technologies. Also, understand the concept of Managed Databases.

INTRO Storage

Lay the foundation for all cloud storage topics. Dive into advantages and considerations regarding data storage and cover essential concepts and terminology.

INTRO Sustainability

Understand how cloud sustainability aligns with broader corporate sustainability goals and learn more about the EU Corporate Sustainability Reporting Directive (CSRD).

INTRO Compliance

How to be compliant in the cloud? Learn all about the benefits and challenges of compliance, sustainability, technical security, and contractual setup in our compliance course.

INTRO Pricing

Demystify cloud pricing with clear explanations and real-world scenarios. Gain insights into Exoscale’s pricing models to make informed decisions for your projects.

Workshop Kubernetes

This hands-on workshop will help you learning the basics of Kubernetes by deploying a microservice application

Workshop CKA Preparation

This hands-on workshop will help you preparing for the Certified Kubernetes Administrator (CKA) certification

Workshop Exoscale Intro

This hands-on workshop will help you deploying a microservice application in Kubernetes using Exoscale’s platform and open-source tools
